# Codex Engineering Principles

Static reference web app for coding principles, software engineering judgment, and effective Codex-assisted development.

## Run

Open [index.html](/C:/Projects/dev-engineering-principles/index.html) in a browser.

## Structure

- [index.html](/C:/Projects/dev-engineering-principles/index.html): app shell and template
- [styles.css](/C:/Projects/dev-engineering-principles/styles.css): visual system and responsive layout
- [app.js](/C:/Projects/dev-engineering-principles/app.js): guide content and search/render logic
